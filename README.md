# Fool2025 - Курсовая работа

## Описание проекта

**Fool2025** — реализация карточной игры "Дурак" на языке C#. Проект представляет собой симулятор игры между двумя игроками с различными стратегиями принятия решений. Проект разработан, как курсовая работа.

## Технологии

- **Язык программирования**: C#
- **Платформа**: .NET 8.0
- **Тип проекта**: Консольное приложение

## Структура проекта

```
Fool2025/
├── Fool2025/
│   ├── Cards.cs          # Определение карт, мастей и правил игры
│   ├── Player1.cs        # Простой игрок с базовой стратегией
│   ├── Player2.cs        # Продвинутый игрок с системой оценки карт
│   ├── Table.cs          # Основная логика игры и управление раундами
│   └── FileName.cs       # Дополнительные классы (MPlayer3)
```

## Основные компоненты

### Cards.cs
Содержит базовые структуры и классы для работы с картами.

### Player1.cs (MPlayer1)
Простой игрок с базовой стратегией:
- Атакует самой младшей картой
- Защищается минимально необходимой картой
- Сортирует карты по рангу

### Player2.cs (MPlayer2)
Продвинутый игрок с системой оценки карт:
- Использует систему штрафов для выбора оптимальной карты при защите
- Учитывает:
  - Ранг карты (`rankK`)
  - Наличие пары в руке (`pairK`)
  - Вероятность наличия карты у соперника (`newCardK`)
- Позволяет настраивать коэффициенты через метод `SetConst()`

### Table.cs (MTable)
Основной класс, управляющий игровым процессом

## Запуск проекта

### Требования
- .NET 8.0 SDK или выше

### Инструкция по запуску

1. Клонируйте репозиторий или откройте папку проекта
2. Откройте решение `Fool2025.sln` в Visual Studio
3. Соберите проект (Build → Build Solution)
4. Запустите проект (F5 или Debug → Start Debugging)

### Запуск из командной строки
```bash
cd Fool2025
dotnet build
dotnet run
```
## Функциональность

При запуске программа:
1. Проводит одну игру с выводом ходов в консоль
2. Проводит 100,000 игр для сбора статистики
3. Выводит результаты:
   - Количество побед первого игрока
   - Количество побед второго игрока
   - Количество ничьих

## Настройка игроков

### Изменение коэффициентов для Player2

В классе `MPlayer2` можно настроить коэффициенты оценки карт:

```csharp
player2.SetConst(rankK, pairK, newCardK);
```
- `rankK` — коэффициент штрафа за ранг карты (рекомендуемый диапазон: 0.4-3.0)
- `pairK` — коэффициент штрафа за наличие пары (рекомендуемый диапазон: 5-20)
- `newCardK` — коэффициент штрафа за вероятность наличия карты у соперника (рекомендуемый диапазон: 2-6)
